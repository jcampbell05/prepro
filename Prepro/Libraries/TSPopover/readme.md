## Require

- iOS 5 -
- ARC
- Add the QuartzCore framework to your project.

## Usage
(see sample Xcode project in `/Demo`)  

## Customize
- background color
- background gradiation (on / off)
- corner radius
- arrow position (Vertical / Horizontal)
- title text 
- title text color
- title text font
